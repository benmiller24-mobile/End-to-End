# Kitchen Cabinet Design Layout Guide

## Cyncly 2020 Design — Frameless Cabinets | Built-In Appliances | Countertops

*A Step-by-Step Decision Guide for Kitchen Designers*

---

# Phase 1: Project Setup & Room Shell

Before placing a single cabinet, you need a clean room shell that accurately represents the physical space. Every measurement error here compounds downstream, so take extra time to get this right.

## 1.1 Create the Project

**Step 1:** Launch Cyncly 2020 Design and select File > New Design.

**Step 2:** Enter the client name, project name, and job site address. Choose the appropriate unit system (Imperial is standard for US kitchen work).

**Step 3:** Select your catalog. For frameless construction, confirm you are loading a frameless-specific catalog (e.g., Eclipse Frameless, or your manufacturer's frameless line). The catalog determines available SKUs, box construction, and hinge boring patterns.

> **Questions to Ask Yourself:**
> - *Am I working from a builder's floor plan, a field measure, or a remodel with existing conditions?*
> - *What unit system does the client expect on their proposal (inches vs. millimeters)?*
> - *Which manufacturer catalog do I need? Is there a frameless-specific catalog, or do I filter within a multi-construction catalog?*
> - *Does the client have a door style or finish already selected, or will that come later?*

| Option | When to Use / Notes |
|--------|---------------------|
| **Manufacturer Frameless** | Best option — SKUs, pricing, and construction rules are pre-configured for frameless boxes |
| **Generic/Custom Catalog** | Use when the manufacturer doesn't have a native 2020 catalog; you'll need to manually configure box specs |
| **Multi-Construction Catalog** | Filter to frameless items only; be careful not to accidentally mix framed and frameless SKUs |

## 1.2 Draw the Room Shell

**Step 1:** Switch to Floor Plan (2D) view. Use the Room Wizard or manually draw walls by clicking to set corner points.

**Step 2:** Enter exact wall lengths from your field measurements. Double-check each wall segment — 2020 shows running dimensions as you draw.

**Step 3:** Set the ceiling height in Design Parameters (typically 96" for standard, 108" or 120" for tall ceilings). This controls how tall cabinet stacks can reach and how high your crown molding lands.

**Step 4:** If the room has soffits, angled walls, or bump-outs, add them now. Soffits are critical because they limit upper cabinet heights and may require shorter uppers or no uppers in certain zones.

> **Questions to Ask Yourself:**
> - *Is my field measure accurate, or do I need to verify? Off-by-an-inch errors cause costly remakes.*
> - *What is the ceiling height? Are there soffits, bulkheads, or dropped ceiling areas?*
> - *Are there any angled walls, bump-outs, or non-90-degree corners?*
> - *Is the floor level, or is there significant slope that affects toe kick and countertop scribing?*
> - *Are there any structural elements (beams, columns, chase walls) that intrude into the room?*

## 1.3 Place Doors, Windows & Openings

**Step 1:** Use the Door tool to place all entry/exit points. Set the exact door width, swing direction, and header height.

**Step 2:** Use the Window tool to place all windows. Enter the sill height (distance from floor to bottom of window frame), window width, and header height.

**Step 3:** Mark any pass-throughs, archways, or half-walls. These define where cabinet runs must start and stop.

> **Questions to Ask Yourself:**
> - *Which direction does each door swing? Will it conflict with any cabinet door or drawer when both are open?*
> - *What is the window sill height? Standard is 36" (counter height) — if lower, I can't run countertop under it. If higher, I may have room for a backsplash shelf.*
> - *Do I want the sink centered under the window? If so, the window placement drives my entire base cabinet layout.*
> - *Are there any pocket doors, barn doors, or bi-folds that need extra clearance?*
> - *Is there an opening to an adjacent room that creates a sight line I should design around (e.g., range hood as focal point from the dining room)?*

| Window Sill Height | Impact |
|---------------------|--------|
| **Below 36"** | Cannot run standard base cabinets or countertop under window; may need open shelving or reduced-depth cabinet |
| **At 36" (counter height)** | Ideal for sink placement; countertop meets sill cleanly |
| **Above 36"** | Full base cabinets below; consider backsplash tile height to fill gap between counter and sill |
| **Floor-to-ceiling** | No base or upper cabinets on that wall segment; plan adjacent runs to compensate for lost storage |

> **Pro Tip:** Place all architectural features before any cabinets. If you add a window after cabinets are placed, 2020 may not automatically adjust — you'll have to delete and re-place cabinets manually.

# Phase 2: Layout Strategy & Work Triangle

The kitchen layout is the single biggest design decision. It determines traffic flow, work efficiency, storage capacity, and the overall feel of the space. Before dragging any cabinets, commit to a layout shape.

## 2.1 Choose the Layout Shape

> **Questions to Ask Yourself:**
> - *How many walls are available for cabinetry? Are any walls shared with appliances only (like a fridge wall)?*
> - *Is there room for an island? Minimum 42" clearance on all four sides (48" preferred for two cooks).*
> - *Is this a galley kitchen in a narrow space, or an open-concept room with island?*
> - *Where does foot traffic enter and exit? The work triangle should not have a traffic path cutting through it.*
> - *Does the client want seating at the island or peninsula? That changes depth and clearance requirements.*

| Layout Shape | When to Use / Notes |
|--------------|---------------------|
| **L-Shape** | Two perpendicular walls. Best for open floor plans with island. Most common and versatile. |
| **U-Shape** | Three walls. Maximum storage. Works well in enclosed kitchens 10'+ wide. |
| **Galley** | Two parallel walls. Efficient for narrow spaces. Keep 42–48" between facing runs. |
| **Single Wall** | Everything on one wall. Common in condos/studios. Pair with island for functionality. |
| **G-Shape** | U-shape with partial fourth wall or peninsula return. Maximum counter space. |
| **L-Shape + Island** | Most popular for open-concept homes. Island adds prep, seating, and storage. |
| **U-Shape + Island** | Only viable in large kitchens (14'+ wide). Risk of feeling closed-in if too tight. |
| **Peninsula** | Like an island but connected to a wall or cabinet run on one end. Saves space vs. island. |

## 2.2 Plan the Work Triangle

The work triangle connects the three primary work centers: sink, range/cooktop, and refrigerator. NKBA guidelines specify each leg should be 4–9 feet, with a total perimeter of 13–26 feet. No traffic path should cross through the triangle.

**Step 1:** Identify where the sink will go. If there's a window, the sink traditionally centers under it. The sink should be between the fridge and the range.

**Step 2:** Identify the range/cooktop position. This should be the focal point of the kitchen — the range hood wall is what people see first. Never place the range in a corner, under a window, or directly next to the fridge.

**Step 3:** Identify the refrigerator position. It should be at the end of a cabinet run (not buried in the middle), ideally the first appliance you reach when entering the kitchen (grocery unloading logic).

**Step 4:** Mentally trace the triangle. Does each leg fall within 4–9 feet? Does anyone walking through the kitchen cross through it?

> **Questions to Ask Yourself:**
> - *Does the plumbing rough-in dictate where the sink must go, or can I move it?*
> - *Where is the gas line or electrical hookup for the range? Moving gas is expensive — is the client willing to pay for it?*
> - *Which wall will people see first when entering from the living/dining area? That's where the range hood should be (focal point).*
> - *If I have an island, will the sink or cooktop be in it? That changes plumbing/electrical and ventilation requirements.*
> - *Is the fridge on the same wall as the entry? It should be the first thing the homeowner reaches with groceries.*

> **Pro Tip:** In an L-shape layout, bias the range and sink toward the corner junction of the L. This naturally tightens the work triangle and keeps the most-used stations close together.

# Phase 3: Base Cabinet Placement

With the layout shape decided and the work triangle mapped, it's time to start placing cabinets. Always place base cabinets first — they anchor appliance positions and drive the upper cabinet layout above them.

## 3.1 Set the Appliance Positions

In 2020 Design, start by placing the three anchor appliances as cabinet items (sink base, range/cooktop space, and refrigerator space). These lock your layout geometry before you fill between them.

**Step 1:** Place the sink base cabinet (typically SB36 for a 36" single-bowl or SB33 for undermount). Center it under the window if applicable. In the frameless catalog, confirm the sink base has the correct interior cutout for your sink type.

**Step 2:** Place the range cabinet or leave a range opening (typically 30" or 36"). In 2020, you can place a range appliance directly, or use a filler/spacer to hold the opening.

**Step 3:** Place the refrigerator cabinet or panel-ready fridge enclosure. For built-in or panel-ready fridges, you'll place an enclosure cabinet (tall or counter-depth) with side panels.

> **Questions to Ask Yourself:**
> - *What size sink did the client select? 30", 33", or 36" base? Undermount, farmhouse/apron front, or top-mount?*
> - *Is the range freestanding, slide-in, or a separate cooktop with wall oven? Slide-in overlaps the counter; freestanding sits between cabinets.*
> - *Is the fridge a standard counter-depth, full-depth, or built-in/panel-ready? Panel-ready needs enclosure panels and REP (Refrigerator End Panels) on both exposed sides.*
> - *What is the fridge width? 30", 36", 42", or 48"? Built-in units can be up to 48" wide.*
> - *Does the range need a filler between it and the adjacent cabinet? Minimum 3" filler next to any tall cabinet.*

| Range/Cooktop Config | When to Use / Notes |
|----------------------|---------------------|
| **Freestanding Range** | Sits between cabinets with its own backguard. Leave exact opening width (typically 30" or 36"). |
| **Slide-In Range** | Overlaps countertop edges by ~1" each side. Opening is the range width. No backguard — needs backsplash behind it. |
| **Separate Cooktop + Wall Oven** | Cooktop drops into a base cabinet cutout (needs 36" wide base). Wall oven goes in a tall oven cabinet. Gives more design flexibility. |
| **Pro/Commercial Range** | Typically 36", 48", or 60" wide. Very heavy — verify floor support. Hood must be at least as wide as the range. |

## 3.2 Fill In the Base Cabinet Run

Working outward from each appliance position, fill the remaining wall space with base cabinets. In 2020, drag cabinets from the catalog and snap them to the wall. Frameless cabinets snap flush without face frame overlap.

**Step 1:** Place landing zone cabinets adjacent to each appliance per NKBA minimums: 24" + 18" flanking the sink, 15" + 12" flanking the range, 15" on the fridge handle side.

**Step 2:** Fill remaining space with a mix of drawer bases (preferred for lower cabinets in modern kitchens), door/drawer combos, and specialty cabinets (trash pull-out, spice pull-out, cookie sheet cabinet).

**Step 3:** Place the dishwasher within 36" of the nearest sink edge. It should be on the side closest to where dishes/glasses are stored in the uppers.

**Step 4:** Check your running dimension against the wall length. You'll likely need fillers to make up the difference.

> **Questions to Ask Yourself:**
> - *Do I have enough landing space next to each appliance? NKBA minimums are the floor, not the goal.*
> - *Does the client prefer drawers or doors on base cabinets? Modern/frameless kitchens trend heavily toward full-extension drawer bases.*
> - *Where does the trash/recycling go? Ideally near the sink. A waste pull-out base is standard.*
> - *Which side of the sink should the dishwasher go? Right-handed users typically prefer DW on the right; left-handed on the left. But prioritize proximity to dish storage above.*
> - *Do I need any specialty base cabinets: lazy susan corner, blind corner pull-out, spice drawer, knife block drawer, or charging drawer?*

## 3.3 Handle the Corners

Corners are the hardest part of any kitchen layout. Frameless cabinets have specific corner solutions that differ from face-frame construction.

> **Questions to Ask Yourself:**
> - *Is this an inside corner (two walls meeting at 90°) or a dead corner?*
> - *Can I use a lazy susan corner base, or does the client prefer a blind corner with pull-out?*
> - *How much filler do I need for the blind corner pull to clear the adjacent door/drawer? Typically 3–5" pull + 3"+ filler on the blind side.*
> - *Does the frameless catalog have a dedicated corner cabinet SKU, or do I need to configure it manually?*
> - *Will the corner countertop be a straight miter or a curved/clipped corner?*

| Corner Solution | When to Use / Notes |
|-----------------|---------------------|
| **Lazy Susan Corner Base** | 36"×36" footprint. Full access to corner. Most storage-efficient. Door or drawer front per catalog. |
| **Blind Corner Base** | Standard base set back 3–5" from the corner. Requires filler strip (3" minimum) on the blind side. Less expensive than lazy susan. |
| **Diagonal Corner Base** | 45° angled front. Takes up more wall space but opens nicely. Less common in frameless lines. |
| **Dead Corner (avoid)** | Inaccessible storage. Only use if absolutely forced by space constraints. Not recommended. |
| **Zero-Corner / Magic Corner** | Premium pull-out system that swings trays out from the corner. Expensive but maximizes access in frameless boxes. |

> **Pro Tip:** In frameless construction, the blind corner filler is non-negotiable. Without a face frame to cover the gap, you'll see raw box edge if you skip it. Always add a 3" minimum filler on the blind side.

# Phase 4: Tall & Pantry Cabinets

Tall cabinets (84", 90", or 96") serve as pantry storage, oven enclosures, and refrigerator surrounds. They are placed after base cabinets because they define the vertical boundaries of your design and influence upper cabinet heights on adjacent walls.

**Step 1:** Determine which tall cabinets the design needs: pantry, oven tall, utility/broom closet, refrigerator enclosure panels.

**Step 2:** Place tall cabinets at the ends of cabinet runs, not in the middle of a base run. They should bookend the kitchen or anchor a dedicated wall.

**Step 3:** For built-in wall ovens, select the correct tall oven cabinet from the frameless catalog. Specify single oven, double oven, or oven/microwave combo. The opening height must match the appliance spec sheet exactly.

**Step 4:** For panel-ready refrigerators, place the refrigerator enclosure panels. Add Refrigerator End Panels (REPs) on both exposed sides of the fridge opening. The hood/panel above the fridge typically matches the surrounding cabinetry.

> **Questions to Ask Yourself:**
> - *What height are my tall cabinets? 84" leaves room for a decorative soffit or space above. 90" or 96" goes to the ceiling for maximum storage.*
> - *If tall cabinets are 84" in a 96" ceiling room, what goes above — open space, a stacking cabinet, or crown molding to the ceiling?*
> - *Is the wall oven a single, double, or combination (oven + microwave)? What are the exact cutout dimensions from the appliance spec sheet?*
> - *Does the fridge need REP panels on both sides? Yes — both exposed sides of any panel-ready fridge get finished end panels.*
> - *Should the hood above the fridge be a matching wood panel or a separate decorative piece?*
> - *Do tall cabinets next to the range need a 3" filler/spacer so the oven door handle clears?*

| Tall Cabinet Height | When to Use / Notes |
|---------------------|---------------------|
| **84" Tall** | Standard. Leaves 12" gap to 96" ceiling. Fill with stacking cabinet, open display, or crown + soffit panel. |
| **90" Tall** | Splits the difference. 6" to ceiling can be covered with a thick crown molding stack. |
| **96" Tall (ceiling height)** | Maximum storage. Goes right to the ceiling. No gap to fill. Most modern/contemporary look. |
| **Custom Height** | Matches non-standard ceiling. Calculate: ceiling height minus any desired crown/reveal at the top. |

> **Pro Tip:** When placing a tall cabinet next to a range, always add a 3" filler between them. This prevents heat damage to the tall cabinet and ensures the range door handle doesn't hit the tall cabinet face.

# Phase 5: Upper Cabinet Placement

Upper cabinets are placed after bases and talls because their widths should align with the base cabinets below for visual symmetry. The bottom of upper cabinets is typically 54" from the floor (18" above a 36" countertop).

## 5.1 Set Upper Heights & Depths

**Step 1:** Set the default upper cabinet mounting height in 2020's Design Parameters. Standard is 54" to the bottom of the upper (18" clear above the counter).

**Step 2:** Choose your standard upper height. Common options: 30", 36", or 42". This is partly functional and partly aesthetic based on ceiling height.

**Step 3:** Set upper cabinet depth. Standard frameless uppers are 12" deep. Some designs use 15" deep uppers for more storage.

> **Questions to Ask Yourself:**
> - *What height uppers work with this ceiling? 42" uppers at 54" mounting = 96" total — perfect for 8' ceilings with no crown. 36" uppers leave room for crown molding or decorative space.*
> - *Does the client want uppers to go all the way to the ceiling (stacked or extended), or leave a gap above?*
> - *Do I need a light rail molding on the bottom of the uppers? It hides under-cabinet lighting and adds a finished look.*
> - *Will any uppers need to be a different height (e.g., shorter cabinets above the fridge, above the microwave, or above a range hood)?*
> - *Is there a soffit that forces a shorter upper cabinet height in certain areas?*

| Upper Height | Ceiling Scenario |
|--------------|------------------|
| **30" Uppers (8' ceiling)** | 12" gap above. Good for crown molding. Most traditional look. |
| **36" Uppers (8' ceiling)** | 6" gap. Crown molding fills to ceiling. Clean finished look. |
| **42" Uppers (8' ceiling)** | Goes to ceiling. No crown (or minimal). Maximum storage, modern aesthetic. |
| **36" Uppers (9' ceiling)** | 18" gap. Use stacking cabinets on top or open display shelf. Popular transitional style. |
| **42" Uppers (9' ceiling)** | 12" gap. Crown molding to ceiling or leave open. |
| **Stacked (any ceiling)** | Standard upper + shorter cabinet stacked on top. Two-tone or glass upper is popular. Reaches any ceiling height. |

## 5.2 Place Uppers Wall by Wall

**Step 1:** Start above the sink. Typically this is a window, so no upper cabinet here. If there's no window, place a standard upper centered on the sink base below.

**Step 2:** Place the range hood or ventilation cabinet above the range. The hood should be at least as wide as the cooktop/range (wider by 3" per side for a grander look). Mount 30–36" above a gas range, 24–30" above electric.

**Step 3:** Flank the range hood with symmetrical upper cabinets. These should be the same width for visual balance. This is the focal wall — symmetry matters most here.

**Step 4:** Fill remaining upper runs, aligning stiles and edges with the base cabinets below wherever possible.

**Step 5:** Above the fridge, place a short upper (12" or 15" tall) or a matching panel depending on your tall cabinet strategy.

> **Questions to Ask Yourself:**
> - *Does the range hood exhaust to the outside, or is it recirculating? Ducted hoods need a chase or soffit for the duct run.*
> - *Is the hood a wood hood (matching cabinetry), a stainless steel chimney hood, or a built-in insert? Wood hoods come from the cabinet catalog; steel hoods are appliance items.*
> - *Are the cabinets flanking the hood the same width? They must be for symmetry. If the wall is asymmetric, use fillers to center the hood.*
> - *Do any upper cabinets have glass doors? Common on the range wall flanking cabinets for display.*
> - *Should I use open shelving anywhere instead of upper cabinets? Mount open shelves at 20" above counter for access.*

| Range Hood Option | When to Use / Notes |
|-------------------|---------------------|
| **Wood Hood (cabinet catalog)** | Matches cabinetry. Comes in chimney style or mantle style. Select from frameless catalog — order as a cabinet SKU. |
| **Stainless Chimney Hood** | Placed as an appliance in 2020. Set width to match or overhang the range. Modern/industrial look. |
| **Under-Cabinet Hood** | Mounts beneath an upper cabinet. Saves space but lower CFM. Place the upper cabinet first, then mount hood below. |
| **Downdraft Vent** | Rises from behind cooktop. No hood needed above. Allows upper cabinets over the cooktop (unusual but possible). |
| **Hood Insert + Custom Surround** | Fan insert goes inside a custom-built wood surround. Maximum design flexibility. Coordinate with the surround vendor. |

> **Pro Tip:** Align major vertical lines between upper and base cabinets. Where two base cabinet stiles meet, the upper cabinet stile above should be at the same horizontal position. This creates visual order — even when exact width matching isn't possible.

# Phase 6: Island & Peninsula Design

If the layout includes an island or peninsula, design it after the perimeter cabinets are in place. The island must maintain proper clearances to all surrounding runs and provide a functional work surface.

## 6.1 Island Sizing & Clearance

**Step 1:** Measure the available floor space between opposing cabinet runs. Subtract minimum clearances (42" each side, 48" preferred, 56" if the fridge is behind seating).

**Step 2:** Determine island depth: 24" (single row of base cabinets + 12–15" overhang for seating) = 39" standard. For back-to-back cabinets, 50" depth.

**Step 3:** Determine island length. Keep under 65" x 128" if you want a single slab of countertop with no seam. Verify with your stone fabricator.

**Step 4:** In 2020, place the island using the catalog's island base cabinets or by creating a cabinet group and positioning it in the floor plan.

> **Questions to Ask Yourself:**
> - *Can I fit an island with 42" clearance on all sides? If not, should I switch to a peninsula (3-sided, minimum 36" clearance)?*
> - *Is the island for prep only, or does it include seating? Seating needs 12–15" overhang and 24" per person.*
> - *Will the island contain a sink, cooktop, or dishwasher? Plumbing in the island adds cost and may require floor access.*
> - *Will the countertop be a single slab or require a seam? Stay under 65" x 128" for one slab.*
> - *Does the client want a waterfall edge on one or both ends? This affects countertop material quantity and fabrication.*
> - *Should the island have a different countertop material or finish than the perimeter (e.g., butcher block island with quartz perimeter)?*
> - *Are there any electrical outlets required in the island? Code typically requires at least one outlet per NEC.*

| Island Configuration | When to Use / Notes |
|----------------------|---------------------|
| **Prep Island (no seating)** | 24" deep base cabinets. All four sides accessible. Purely functional. |
| **Seating Island (one side)** | 24" bases + 12–15" overhang on one long side. Standard 39" total depth. Most common. |
| **Back-to-Back Cabinets** | Two rows of bases facing opposite directions. 50" deep. Great for large kitchens needing lots of storage. |
| **Waterfall End(s)** | Countertop wraps down one or both short ends. High-end modern look. Increases slab material cost. |
| **Peninsula (attached)** | One short end connects to a wall or cabinet run. Only needs 3-sided clearance (36" min). Space-saving alternative. |
| **T-Shape or L-Shape Island** | Non-rectangular island for large or oddly shaped rooms. Complex to detail in 2020 — may need manual adjustments. |

## 6.2 Island Cabinet Placement in 2020

**Step 1:** Place base cabinets for the island using the island-specific items from your frameless catalog, or convert standard bases to island orientation.

**Step 2:** Add finished end panels (FEPs) to all exposed island sides. Unlike perimeter cabinets (which may have only one exposed end), every side of an island is visible.

**Step 3:** If the island has seating, set the countertop overhang in the countertop properties (coming in Phase 9). Add support brackets if overhang exceeds 10".

**Step 4:** Apply decorative details: furniture base/legs, decorative end panels, corbels, or open shelving cubbies on the back side.

> **Pro Tip:** Every face of an island cabinet is visible. Budget for finished end panels on both sides, and consider decorative back panels if the back of the island faces the living area.

# Phase 7: Built-In Appliance Integration

Built-in and panel-ready appliances are a hallmark of high-end frameless kitchen design. Each built-in appliance interacts with the cabinetry differently and has specific clearance, panel, and trim requirements.

## 7.1 Panel-Ready Refrigerator

**Step 1:** Place the refrigerator appliance in the opening you created during base cabinet placement.

**Step 2:** Add Refrigerator End Panels (REPs) on both exposed sides. These are ordered as cabinet items from the frameless catalog — they match the door style and finish.

**Step 3:** Place the panel above the refrigerator. This is typically a framed panel or short door that matches adjacent cabinetry and fills the space to the soffit/ceiling.

**Step 4:** Verify the fridge door opens at least 170° for shelf removal. Check that the REP panel and any adjacent wall/corner doesn't block the door swing.

> **Questions to Ask Yourself:**
> - *Is this a panel-ready or stainless-look built-in? Panel-ready needs custom door panels from the cabinet manufacturer.*
> - *What is the exact fridge depth? Counter-depth (24"), standard depth (30"), or full built-in (24" with panels)?*
> - *Do the REP panels need to be flush with adjacent cabinetry, or is a slight reveal acceptable?*
> - *Does the fridge have a bottom freezer with a pullout drawer? Verify clearance for the drawer to extend fully.*
> - *Is there a water/ice line? Verify supply line routing doesn't interfere with adjacent cabinet interiors.*

## 7.2 Built-In Wall Oven & Microwave

**Step 1:** Select the correct tall oven cabinet from the frameless catalog. Verify the interior opening dimensions match the appliance spec sheet.

**Step 2:** Place the appliance in 2020 using the Built-In Oven tool or by inserting the appliance item into the tall cabinet.

**Step 3:** For combination oven + microwave towers, verify both cutout heights and the total stack height within the cabinet.

**Step 4:** Add a 15" landing surface adjacent to the oven (within 48" maximum). This is where hot dishes land when coming out.

| Oven Tower Config | When to Use / Notes |
|-------------------|---------------------|
| **Single Wall Oven** | One oven in a 30" or 27" wide tall cabinet. Upper section can be a warming drawer or storage. |
| **Double Wall Oven** | Two full ovens stacked. Requires 84"+ tall cabinet. Ideal for clients who entertain frequently. |
| **Oven + Microwave Stack** | Wall oven below, built-in microwave above (or vice versa). Most space-efficient combo. |
| **Oven + Speed Oven** | Wall oven below, speed/convection microwave above. Premium option for serious cooks. |
| **Steam Oven Combo** | Wall oven + steam oven stack. Growing in popularity. Check specific brand cutout requirements. |

## 7.3 Built-In Dishwasher

**Step 1:** Place the dishwasher within 36" of the nearest sink edge. In 2020, replace a 24" base cabinet with the dishwasher appliance.

**Step 2:** For panel-ready dishwashers, order the matching door panel from the frameless catalog. The dishwasher front will match adjacent cabinetry.

**Step 3:** Verify the dishwasher door clears the adjacent cabinet doors and drawers when fully open. Check island or peninsula clearance if the DW faces an island.

## 7.4 Built-In Microwave Options

> **Questions to Ask Yourself:**
> - *Where should the microwave go? In the oven tower, above the range (OTR), in a dedicated base cabinet drawer, or in a custom trim kit?*
> - *Does the client want the microwave hidden (in a base cabinet with a pocket door or a dedicated microwave drawer)?*
> - *If placing an over-the-range (OTR) microwave, does it replace the range hood? OTR microwaves have a built-in vent but lower CFM than a dedicated hood.*
> - *Is a microwave drawer preferred? These install in a base cabinet below the counter for ADA accessibility and clean sightlines.*

| Microwave Placement | When to Use / Notes |
|---------------------|---------------------|
| **In Oven Tower** | Built-in trim kit above or below the wall oven. Cleanest look. Keeps counters clear. |
| **Over-the-Range (OTR)** | Replaces range hood. Saves space but reduces ventilation power. Not recommended for serious cooks or gas ranges. |
| **Microwave Drawer** | Installs in a 24"+ wide base cabinet. Opens from the top like a drawer. ADA-friendly and kid-friendly. |
| **Countertop + Trim Kit** | Built into a cabinet shelf opening with a trim kit. More affordable than a built-in unit. |
| **Dedicated Upper Cabinet** | Open-shelf upper with a built-in microwave. Common in transitional designs. |

# Phase 8: Fillers, End Panels & Decorative Trim

Fillers and end panels are where frameless cabinets separate from framed. Without a face frame to hide gaps, every joint and exposed edge must be intentionally finished. This phase is critical for a professional result.

## 8.1 Fillers

**Step 1:** Add filler strips at every wall-to-cabinet junction where the cabinet doesn't meet the wall squarely. A 3" filler is standard; wider if the wall is out of square.

**Step 2:** Add fillers next to all appliances: minimum 3" between a range and a tall cabinet, and any manufacturer-specified clearance.

**Step 3:** Add fillers at blind corners: minimum 3" on the blind side so the door/drawer clears when opened.

**Step 4:** Review the design in 2020's 3D view. Zoom into every filler location. Do they look proportional? Is the filler finish-matched to the adjacent cabinet?

> **Questions to Ask Yourself:**
> - *Is the filler strip the same material and finish as the cabinets, or a contrasting accent?*
> - *Are my walls plumb and square? If not, I may need wider fillers (up to 6") to scribe against the wall.*
> - *Does the filler need to extend from countertop to floor (base filler) or countertop to upper bottom (splash filler)?*
> - *Do I have any inside corner fillers where two cabinet faces meet at 90°? These prevent door/drawer handle conflicts.*

## 8.2 End Panels & Skin Panels

**Step 1:** Add finished end panels to every exposed cabinet side. This includes: cabinet ends at doorways, cabinet ends at appliance openings, both sides of the refrigerator, all island faces, and any cabinet visible from a walkway.

**Step 2:** Specify panel type: slab (flat panel matching door), shaker profile (matching door profile), or flush/scribed (for tight-to-wall installations).

**Step 3:** Verify panel depth matches the cabinet box depth plus the door thickness. Frameless panels sit flush with the door face.

## 8.3 Crown Molding, Light Rail & Toe Kick

**Step 1:** Add crown molding to the top of upper and tall cabinets. In 2020, select the molding profile from the catalog and apply it to the cabinet run. Specify whether it returns to the wall at run ends.

**Step 2:** Add light rail molding to the bottom of upper cabinets. This finishes the underside and hides LED strip lighting.

**Step 3:** Verify toe kick material: matching cabinet finish, stainless steel, or recessed black. Frameless toe kicks are typically 4½" tall.

> **Questions to Ask Yourself:**
> - *Does the crown molding go all the way to the ceiling, or is there a reveal? Modern designs often skip crown entirely for a clean, slab look.*
> - *Is the crown profile simple (cove or ogee) or stacked (multiple layers)? Stacked crown needs more vertical space.*
> - *Does the light rail match the door style or is it a simple strip?*
> - *What color/material is the toe kick? Matching the cabinet looks built-in; recessed black makes the cabinet appear to float.*

| Crown Strategy | When to Use / Notes |
|----------------|---------------------|
| **Crown to Ceiling** | Molding fills the gap between upper top and ceiling. Traditional/transitional. Requires a sub-molding or filler board if gap > 3". |
| **Crown as Cap (with gap)** | Molding sits on top of the cabinet with open space above. Casual/farmhouse look. Collects dust. |
| **No Crown (flat top)** | Clean modern look. Upper cabinets must have a finished top if visible. Typical in frameless/Euro designs. |
| **Stacked Crown** | Multiple molding profiles layered for a grand, furniture-like look. Requires taller filler space above uppers. |

# Phase 9: Countertop Generation & Editing

In Cyncly 2020, countertops can be auto-generated from the base cabinet run and then edited for overhangs, edge profiles, and material. This is typically one of the last steps before rendering.

**Step 1:** Use 2020's Auto-Generate Countertop feature: select all base cabinets in a run and generate the countertop. It will snap to the cabinet perimeter.

**Step 2:** Set the countertop material: quartz, granite, marble, butcher block, laminate, concrete, etc. This affects rendering appearance and, in some catalogs, pricing.

**Step 3:** Edit the edge profile: eased, beveled, bullnose, ogee, waterfall, mitered. Select the profile that matches the overall design style.

**Step 4:** Adjust overhangs: standard is 1" to 1½" past the cabinet face on front and open ends. Seating overhangs are 12–15".

**Step 5:** Edit the countertop shape for sink cutouts, cooktop cutouts, and any custom shapes (clipped corners, curves at peninsulas).

**Step 6:** Add a backsplash if desired. In 2020, you can generate a 4" or full-height backsplash from the countertop tool or apply it as a separate material on the wall.

> **Questions to Ask Yourself:**
> - *What countertop material did the client select? Each material has a maximum slab size that affects seam placement.*
> - *Is the edge profile consistent throughout, or does the island get a different edge than the perimeter?*
> - *Where will seams be? Plan seams at natural break points (behind the sink, at an inside corner) to minimize visibility.*
> - *Does the countertop overhang on the island need corbels or a steel support bracket for spans over 10"?*
> - *Is there a waterfall edge on the island? This is a separate countertop piece in 2020 — add it as an end panel material.*
> - *What backsplash height? 4" standard, full-height to the upper cabinets (18"), or slab backsplash (same material as counter)?*

| Countertop Material | Considerations |
|---------------------|----------------|
| **Quartz (engineered)** | Non-porous, consistent color, low maintenance. Max slab ~63" x 126". Most popular choice. |
| **Granite (natural stone)** | Each slab unique. Needs annual sealing. Max slab ~60" x 120" typical. |
| **Marble** | Luxurious but porous and etches easily. Best for areas not exposed to acids. Popular for islands/baking stations. |
| **Butcher Block** | Warm wood surface. Needs oiling. Great for island prep areas. Not recommended near sink (water damage). |
| **Laminate** | Most affordable. Wide range of patterns including stone-look. Edges are the giveaway — use a built-up edge. |
| **Porcelain Slab** | Ultra-thin, heat-resistant, scratch-resistant. Large format slabs. Growing in popularity. |
| **Concrete** | Custom-poured or prefab. Industrial look. Heavy. Needs sealing. Can crack if not reinforced properly. |

| Edge Profile | Design Style |
|--------------|--------------|
| **Eased/Straight** | Minimal flat edge with slight roundover. Modern/contemporary. Most popular for frameless kitchens. |
| **Beveled** | Angled cut at the top edge. Transitional. Adds subtle detail without being ornate. |
| **Bullnose (full or half)** | Rounded edge. Traditional/transitional. Softens the look. Half bullnose is more common today. |
| **Ogee** | S-curve profile. Traditional/formal. Adds visual weight and elegance. |
| **Mitered** | Thick-look edge made by mitering two pieces. Creates 3"+ thick appearance. Modern luxury. |
| **Waterfall** | Countertop cascades down one or both ends to the floor. Contemporary statement. Premium cost. |

# Phase 10: Review, Validate & Present

Before showing the design to the client, run through a complete validation checklist. Catching errors now is infinitely cheaper than catching them during installation.

## 10.1 Dimensional Validation

**Step 1:** Switch to Floor Plan (2D) view and verify all running dimensions. Each wall's cabinet run + fillers + appliance openings should total the exact wall length.

**Step 2:** Check clearances: 42"+ walkways, 48"+ between parallel runs with two cooks, 56" behind island seating facing a fridge.

**Step 3:** Verify the work triangle: each leg 4–9', total 13–26', no traffic crossing through it.

**Step 4:** Verify all appliance landing zones meet NKBA minimums: sink (24" + 18"), range (15" + 12"), fridge (15" handle side), oven (15" within 48").

## 10.2 3D Visual Review

**Step 1:** Switch to 3D Perspective view. Walk around the entire kitchen slowly. Check for visual balance, symmetry on the range wall, and anything that looks off.

**Step 2:** Look at the range hood wall straight-on. Are flanking cabinets equal width? Does the hood overhang the range by 3" per side? Is this the best-looking wall in the kitchen?

**Step 3:** Check all door and drawer operation in 3D. Open every door and drawer. Do any collide with each other, with appliance doors, or with room doors?

**Step 4:** Verify end panels are on all exposed sides. Check the island from all four directions.

## 10.3 The 25 Things to Never Do

Run through this final checklist to catch common mistakes:

| Never Do This | Why |
|---------------|-----|
| **Range in a corner** | Poor access, grease on adjacent walls, fire hazard |
| **Range under a window** | Curtains, grease on glass, code violations in some areas |
| **Range next to fridge (no buffer)** | Need 9"+ minimum buffer; designers recommend 15–24" |
| **Range next to tall cabinet (no filler)** | Heat damage and handle clearance issues |
| **Sink in a corner** | Uncomfortable to stand at, limits faucet options |
| **DW at right angle to sink** | Awkward loading; water drips across the floor |
| **DW on wrong side of sink** | Should be on the side closest to dish/glass storage in uppers |
| **Fridge buried mid-run** | Should be at the end of a run for door swing and grocery access |
| **Dead corner with no access** | Wasted space; always use a lazy susan, blind corner pull-out, or magic corner |
| **Missing end panels** | Exposed cabinet box sides look unfinished and unprofessional |
| **Missing filler at blind corner** | Doors/drawers won't open; raw edge visible in frameless |
| **Upper/base misalignment** | Vertical lines should align between upper and base cabinets |
| **Walkway under 36"** | Below code minimum; 42" is the real minimum for comfort |
| **Walkway over 50"** | Kitchen feels disconnected; everything is too far apart |
| **Traffic through work triangle** | The triangle is a workspace, not a hallway |

## 10.4 Generate Deliverables

**Step 1:** Generate a photorealistic render from 2020's rendering engine. Set up 2–3 camera angles: the range wall straight-on (hero shot), a three-quarter room view, and the island/seating area.

**Step 2:** Export the floor plan with dimensions for the installer.

**Step 3:** Generate the cabinet specification list (CSV or report) with all SKUs, quantities, and pricing.

**Step 4:** Generate an elevation drawing for each wall showing cabinet heights, widths, and appliance positions.

**Step 5:** Review the proposal/order form. Verify every line item, especially fillers, end panels, and molding — these are the most commonly missed items in cabinet orders.

> **Pro Tip:** The most commonly forgotten items on a cabinet order are: fillers (wall junctions, appliance buffers, blind corners), end panels (every exposed side), and molding returns (where crown or light rail terminates at a wall). Double-check these specifically before submitting the order.
